# 🎬 Movie Genre Classification using Machine Learning

A machine learning project developed during my internship at CodSoft.

This project predicts the genre of a movie based on the movie description using Natural Language Processing (NLP) and Machine Learning.

Built and trained entirely in Google Colab with an interactive user interface for real-time prediction.

---

## 📌 Project Overview

Movie genre classification helps automatically identify the category of a movie from its storyline or description.

This project uses text preprocessing and machine learning algorithms to analyze movie descriptions and predict genres accurately.

---

## 🚀 Features

✅ Movie genre prediction from description  
✅ NLP text preprocessing

✅ Multiple ML algorithms implementation

## 📊 Workflow

### 1. Dataset Loading
Imported movie dataset provided by CodSoft.

### 2. Data Preprocessing
- Cleaned movie descriptions
- Prepared text data

### 3. Feature Extraction
Used TF-IDF Vectorizer to convert text into numerical features.

### 4. Model Training
Trained ML algorithms on movie descriptions.

### 5. Model Evaluation
Compared prediction accuracy.

### 6. Model Saving
Saved trained model using Joblib.

### 7. User Interface
Built interactive prediction UI using Gradio in Google Colab.
